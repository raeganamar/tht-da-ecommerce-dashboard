## Overview
This project analyzes revenue quality beyond growth metrics, focusing on:
- Revenue concentration by country
- Order status contribution
- Return rate impact
- Customer repeat behavior

The goal is to evaluate sustainability and profitability risk, not just revenue growth.

---

## Key Insights
- Total Net Revenue: $8.13M
- Repeat Customer Rate: 37.74%
- Return Rate: ~10%
- Revenue highly concentrated in China

Revenue growth alone does not guarantee sustainability. Return impact and market concentration must be monitored.
